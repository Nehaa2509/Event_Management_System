# Event Management System

A robust, web-based **Event Management System** built using **Django** and **SQLite** (or PostgreSQL) designed to seamlessly create, manage, and discover local and virtual events. The application features comprehensive user role management, secure registration, and automated ticketing tracking.

---

## 🚀 Features

### 👤 User Roles & Authentication
* **Admin:** Full control over users, event categories, and platform-wide metrics.
* **Organizer:** Dedicated dashboard to create, update, and delete events, as well as track real-time ticket registrations.
* **Attendee:** Browse upcoming events, filter by category/location, register for tickets, and view booking history.

### 📅 Event Management
* Full CRUD operations for events (Title, Description, Date, Time, Venue/Virtual Link, Banner Image, and Capacity).
* Automated status handling (Upcoming, Ongoing, Passed).

### 🎟️ Ticketing & Booking System
* Real-time seat/capacity tracking to prevent overbooking.
* Unique Booking ID generation upon successful registration.
* Easy booking cancellation workflow.

---

## 🛠️ Tech Stack

* **Backend:** Python / Django (Web Framework)
* **Database:** SQLite (Default development database)
* **Frontend:** HTML5, CSS3, Bootstrap (Responsive Design)
